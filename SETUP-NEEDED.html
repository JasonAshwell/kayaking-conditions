⚓ Setup Required

Your Sea Kayaking Conditions app needs a free API key to access tide data

Why do I need this?

  • Completely Free: 1000 requests/month forever (no credit card)
  • Quick Setup: Takes just 2 minutes
  • Accurate Data: Global tide predictions for any location
  • One-Time Setup: Do it once, works forever
1

Register for Free Account

Visit WorldTides and create a free account:

Register Free Account →

Fill in your email, choose a password, and verify your email.

2

Get Your API Key

After verifying your email:

Get API Key →

Log in and find your API key under "API Keys" section.

It looks like: 135660c8-6da3-4b8f-baf2-eb3f3c56b6c9

3

Add Key to Config File

Open this file in a text editor:

js/utils/config.js

Find these lines (around lines 7-9):

apiKey: '',
...
useFreeMode: true

Change to:

apiKey: 'YOUR-ACTUAL-KEY-HERE',
...
useFreeMode: false

Save the file.

4

Refresh and Use!

Refresh your browser and start using the app:

Open App →

✓ That's it! You now have unlimited access to tide data for planning your kayaking trips.

Having Trouble?

Read the detailed guide: GET-API-KEY.md

Why Not Just Use UKHO?

The UKHO API you have doesn't work in browsers due to CORS restrictions (browser security). WorldTides is designed for web apps and works perfectly!
